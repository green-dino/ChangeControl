
## Simplest Python Program
*Supports Python Version 2.x and 3.x*

```python
# Simple print to standard out
print("Hello World")

# Create simple variables / actually objects
valueInt = 25
print(type(valueInt))
print(valueInt)

valueFloat = 2.500
print(type(valueFloat))
print(valueFloat)

msgStr = "Hello World"
print(type(msgStr))
print(msgStr)
```
